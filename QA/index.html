常见问题

注意事项

声明

  1. 本站所有商品代买服务都购自知名品牌官网或合法授权商家。若需要代买的物品或商家我们不了解，我们保留谢绝服务的权利。
  2. 为了保护个人信息，我们不提供购物单据。如果实在有需要，请务必提前告知，我们可以提供收据电子档，但会遮盖任何隐私信息。
  3. 您与我们确认下单后，表示您已阅读我们的代买条款，同意条款内容和最终计价。
  4. 我们保留随时修改注意事项的权利

关于收货时间

确认最终订单并下单成功后，

* 代购服务根据各品牌商品不同，一般14-21个工作日可收到货物
* 限量购物品一般3-7个工作日可收到货物
  • 注：以上收货时间仅供参考，实际收货收货时间长短还受到海外商家的发货时间（代购服务）以及海关清关速度影响。我们会尽量缩短等待时间，但是若因这些不可抗因素导致延误时，还请大家见谅。对于赶时间的买家请慎重考虑，或提前与我们联系告知你的特别需求。

关于运费

运费根据将根据重量计算，美国的重量计量单位是磅（Lb），一磅约为0.454公斤。所有代购商品，由买家自行支付运费。建议买家善用手头类似物品估算商品重量，并根据这个表格估算运费。最终确认订单时，我们也会估算运费供买家参考。买家最终支付运费以实际运费支出为准。

限量购物品运费由我们吸收。

关于关税

  1. 所有代买服务产生的关税一律由买家承担
  2. 所有限量购物品产生的关税目前由我们吸收

关于取消订单

  1. 确认订单后，若在支付前需要取消订单，请及时联系
  2. 确认订单后并完成支付后，订单会立刻执行，若订单已执行，则无法取消
  3. 若物品金额在确认订单后，完成支付前出现变化，我们会将相关变化予以通知，若买家认同相关变化方会继续执行订单

退换货规则

  1. 贴心代购服务一旦下单后无法更改订单，请提前确认好物品的款式，尺寸，颜色等是否正确，如需协助请询问。
  2. 除非我们在执行您的代买指示时出错，否则代购费用不予退还。
  3. 由于跨境购物环节多，流程较长，且商品一般均由海外商家原装发货，若签收时发现物品损坏和质量问题，可在物品实际签收的7个工作日内申请退货。详情请看退货流程。
  4. 退货时，请确保商品吊牌，防伪标签，以及各种包装尽量完整。否则可能影响退货。
  5. 以下情况下不提供退货服务：
    • 除非小葡萄代购出货的错误，否则不接收非质量问题为由的退货。（实际尺寸与标的尺寸的偏差，以及照片和图片的色差等属于非质量问题）
    • 由于非正常使用导致产品出现质量问题
    • 商品外观受损，吊牌标签被剪掉或遗失
    • 美国商家本身不提供退货服务（如ebay,亚马逊第三方卖家，大多数奢侈品，以及不提供退货的清仓物品），若不清楚哪些商家提供退货服务，请提前与我们确认。
    • 贴身衣服等只要拆开包装就不能退了

退货流程说明

  1. 退货时，请将商品名称，订单日期以及退货原因告知我们，并向我们索要退货邮寄地址
  2. 在寄回商品时，请确保商品包裹良好，否则可能影响您的退款
  3. 我们将尽全力协助退款，但最终退款结果取决于美国商家，一般由于质量问题发起的退款都会获得全额退款，有些品牌会按比例收取处理费用。全额退款仅包括买家支付的费用商品费用。
  4. 退款时间各个品牌都不同，一般从所退商品抵达商家仓库算起，约需3-6周的时间。我们只有在收到商家的时间退款后，才会将退款发回买家。

不接受代购订单的物品包括但不限于以下物品

  1. 香水、指甲油、独立电池、充电宝、处方药；
  2. 超重物品（单个物品大于20kg）、超大物品；
  3. 剃须刀的清洗液；
  4. 易燃易爆危险品、喷雾/气压罐类物品、易腐蚀品、管制刀具、毒品、军火、文物古玩；
  5. 宠物饲料、各种肉制品、西洋参、燕窝、花旗参、烟、酒等；
